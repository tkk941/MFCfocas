This document provides the information about "Fwlib32.cs" and "Fwlib32.vb".
Please read this document before using these files.
Since the functions of all FOCAS2 is not defined, add the function not defined
in this file.

------------------------------------------------------------------------------

  - FWLIB32.CS : The header file for Visual C# .NET.
  - FWLIB32.VB : The header file for Visual Basic .NET.

These files are samples for Visual C# .NET or Visual Basic .NET.

Maximum control axes number and Maximum spindle number are defined as follows
in these files.

  - Maximum control axes number : 32 axes
  - Maximum spindle number      : 8
